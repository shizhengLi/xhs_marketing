# 小红书热点监控工具 - 前端

## 安装依赖

```bash
npm install
```

## 运行开发服务器

```bash
npm run dev
```

## 构建生产版本

```bash
npm run build
```

## 技术栈

- React 18
- TypeScript
- Vite
- React Router
- Ant Design
- Zustand
- Axios

## 项目结构

- `src/components` - 通用组件
- `src/pages` - 页面组件
- `src/services` - API服务
- `src/store` - 状态管理
- `src/hooks` - 自定义Hooks
- `src/types` - TypeScript类型定义
- `src/utils` - 工具函数
